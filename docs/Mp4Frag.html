 Configuration options.
Properties
Name Type Attributes Description
hlsBase String <optional>
 Base name of files in fmp4 m3u8 playlist. Affects the generated m3u8 playlist by naming file fragments. Must be set to generate m3u8 playlist.
hlsListSize Number <optional>
 Number of segments to keep in fmp4 m3u8 playlist. Must be an unsigned int ranging from 2 to 10. Defaults to 4 if hlsBase is set and hlsListSize is not set.
bufferListSize Number <optional>
 Number of segments to keep buffered. Must be an unsigned int ranging from 2 to 10. Independent of HLS settings.
callback function <optional>
 Function to be called when segments are cut from pipe. Must be able to pass 1 parameter that will contain segment buffer.
