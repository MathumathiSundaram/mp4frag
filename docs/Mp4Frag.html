Properties
Name Type Attributes Description
hlsBase String <optional>
 Base name of files in fmp4 m3u8 playlist. Affects the generated m3u8 playlist by naming file fragments. Must be set to generate m3u8 playlist.
hlsListSize Number <optional>
 Number of segments to keep in fmp4 m3u8 playlist. Must be an integer ranging from 2 to 10. Defaults to 4 if hlsBase is set and hlsListSize is not set.
hlsListInit Boolean <optional>
 Indicates that m3u8 playlist should be generated after init segment is created and before media segments are created. Defaults to false.
bufferListSize Number <optional>
 Number of segments to keep buffered. Must be an integer ranging from 2 to 10. Not related to HLS settings.
